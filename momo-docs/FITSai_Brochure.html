1. Chat & AI Intelligence

The core of FITSai - an intelligent assistant that learns and adapts

Smart AI Routing User

FITSai uses a smart routing system that automatically selects the best AI model for each question, optimising for speed, cost, and quality.

Query TypeModelWhereSpeed
Simple questionsllama3.1:8b via OllamaLocal GPU1-3s
Coding questionsdeepseek-coder-v2 via OllamaLocal GPU2-5s
Knowledge base answersllama3.1:8b (local first, GPT-4o-mini fallback)Local / Cloud1-4s
Complex reasoningGPT-4o-mini (OpenAI)Cloud1-3s
Image analysisGPT-4o-mini (OpenAI)Cloud2-4s

How it works: When you ask a question, FITSai analyses the complexity (word count, keywords, context) and routes it to the most appropriate model. Simple questions run on the local GPU for free, while complex reasoning and image analysis use cloud AI. If the local model is unavailable, everything gracefully falls back to OpenAI.

Knowledge Base (RAG) User

FITSai searches your company's uploaded documents before answering, so responses are grounded in real company knowledge.

  • Upload PDF, DOCX, TXT, or image files (up to 50MB)
  • Documents are automatically chunked and indexed for semantic search
  • Answers include confidence scores and source references
  • Previously approved answers are cached to save costs
Cost Savings: When you give a thumbs-up to an answer, it's saved in the learned answer database. Similar future questions reuse this answer at zero API cost. Local model routing further reduces costs by handling simple queries on-device.

Self-Correction & Quality Grading User

Every response from a local AI model is automatically quality-checked in the background by GPT-4o-mini:

  • Local model answers are graded for accuracy and completeness
  • Poor-quality answers are corrected and saved to the knowledge base
  • Future similar questions retrieve the corrected answer via RAG
  • Thumbs-down feedback triggers immediate re-grading (bypasses sampling)

This means local model answers are never unchecked — they go through the same quality pipeline as cloud answers, just asynchronously.

Streaming Responses User

Responses appear in real-time as they're generated, token by token - no waiting for the full answer. Uses Server-Sent Events (SSE) for smooth, live text delivery. Full conversation thread context is maintained so FITSai understands your entire discussion.

Conversation Threading & Sharing User

Organise your conversations into separate threads by topic. FITSai remembers context within each thread, so you can switch between projects without losing your place.

  • Create new threads or continue existing ones
  • Search across all your conversation history
  • Soft-delete conversations (recoverable, no data loss)
  • Export chats as PDF, JSON, or TXT
  • Share conversations — copy a formatted thread to clipboard for sharing with colleagues

Chat Templates User

Pre-defined prompts for common tasks, accessible via Ctrl+T. Templates help you get consistent, well-structured responses for recurring workflows like daily standups, code reviews, or ticket summaries.

Feedback Learning Loop User

Every response includes thumbs-up/thumbs-down buttons. Your feedback directly improves FITSai:

  • Thumbs Up: Answer is saved for future reuse (+10 XP for you)
  • Thumbs Down: Answer is flagged for improvement
  • High-confidence matches (85%+) reuse approved answers instantly

2. Smart Colleague Intelligence

FITSai learns how you work, detects frustration, and connects you with team knowledge

Morning Briefing User

Start your day with a personalised briefing card that appears when you first open FITSai:

  • Personalised greeting based on time of day and your streak
  • Pending reminders due today
  • Recent achievements and XP progress
  • Follow-up on unresolved issues from the past 7 days
  • Daily tip tailored to your usage patterns

Frustration Detection User

FITSai detects when you're getting frustrated and automatically adjusts its tone:

  • Analyses keywords, CAPS LOCK, punctuation patterns, and repeated thumbs-down feedback
  • Three frustration levels: mild, moderate, high
  • Empathy is injected into AI context, adapted to your active personality mode
  • High frustration triggers a gentler, more supportive tone without being patronising

Communication Style Adaptation User

FITSai learns your communication preferences from your feedback patterns:

  • Brevity: Learns whether you prefer short or detailed answers
  • Formality: Adapts between casual and formal tone
  • Technical depth: Adjusts jargon level based on your expertise

Preferences are learned automatically from thumbs-up/down feedback — no configuration needed.

Peer Knowledge Linking User

When you ask a question, FITSai checks if a colleague in your department has solved a similar problem before. If so, it mentions who solved it and what the solution was, connecting team knowledge across conversations.

Proactive Nudges User

FITSai detects when you've asked about the same topic 3+ times and proactively suggests creating a cheat sheet or document from your repeated questions — turning tribal knowledge into permanent documentation.

6. Productivity Tools

Reminders, documents, memory, and search to keep you productive

Reminder System User

Set reminders using natural language and receive popup notifications when they're due.

  • Create: "Remind me to call the client at 3pm" - FITSai parses the time and title
  • Popup: Orange notification appears in top-right corner when due
  • Actions: Mark as Done, Snooze 15 minutes, or Snooze 1 hour
  • Overdue: All overdue reminders keep showing until you dismiss them
  • Recurrence: Set daily, weekly, or monthly recurring reminders

Reminders are checked every 30 seconds in the background, so you'll never miss one while using FITSai.

Document Management User

Upload and manage company documents to build the AI knowledge base.

  • Supported formats: PDF, DOCX, TXT, PNG, JPG (up to 50MB)
  • Documents are automatically chunked, embedded, and indexed
  • FITSai searches your documents when answering questions
  • Document Analysis: Summarise, analyse, extract key info, or memorise documents
  • Document Conversion: Convert DOCX/XLSX to PDF
  • Earn 25 XP for each document uploaded

Memory System User

FITSai remembers important information about you across sessions.

  • Automatically extracts key facts from conversations (40+ recognition patterns)
  • Memories have importance scores that decay over 30 days
  • Semantic search finds relevant memories even with different wording
  • View, search, and manage your memories from the Memory Manager page
  • All AI responses are informed by your stored memories for personalised answers

Chat Search & Export User

Search across your entire conversation history with keyword matching and highlighting. Export any conversation as PDF, JSON, or TXT for record-keeping or sharing.

10. Security

Enterprise-grade authentication and data protection

Authentication

  • JWT Tokens: HS256 algorithm, 7-day expiration
  • Password Hashing: Bcrypt with salt rounds
  • Role-Based Access: User, Manager, Admin roles
  • Level-Based Elevation: Level 46+ gains God Mode access
  • Password Reset: Email token-based forgot password flow

Data Protection

  • SQL injection prevention via ORM parameterised queries
  • Pydantic schema validation on all API inputs
  • CORS configuration for allowed origins
  • WebSocket token validation for voice sessions
  • Document storage outside web root
  • Soft-delete for conversations (no permanent data loss)
  • Voice data not permanently stored on server

11. Quick Start Guide

Get started with FITSai in minutes

For Users

  1. Login - Use your company email and password
  2. Ask a question - Type in the chat box and hit Enter (+5 XP)
  3. Give feedback - Click thumbs up or down on answers (+10 XP)
  4. Upload documents - Add company docs to improve answers (+25 XP)
  5. Check your progress - View level, XP, and achievements on your profile
  6. Set reminders - Ask FITSai "Remind me to..." for natural language reminders
  7. Try voice - Click the avatar to start a voice conversation
  8. Switch personalities - Say "Switch to [mode] mode" for a different vibe
  9. Compete - Check the leaderboard to see your ranking

For Admins

  1. Access God Mode - Click the God Mode icon in the navigation (admin role required)
  2. Manage users - Search, edit, and create user accounts
  3. Award XP - Recognise users with XP bonuses
  4. Monitor system - Check system stats, service health, and API costs
  5. Update roadmap - Change feature statuses from the roadmap panel
  6. View analytics - Check model usage, costs, and feedback stats on the Budget page

Tips for Best Results

  • Be specific: More details = better answers
  • Give feedback: Thumbs up/down improves FITSai and earns you XP
  • Upload documents: The more company knowledge, the better the answers
  • Try different personalities: Find the communication style that works for you
  • Use templates: Press Ctrl+T for pre-built prompts
  • Ask follow-ups: FITSai remembers your conversation context
