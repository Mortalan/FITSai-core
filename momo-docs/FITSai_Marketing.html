Deployment

  • Self-hosted on your own server
  • Runs on a single machine with consumer GPU
  • Docker + systemd — standard Linux stack
  • No cloud dependency for local queries
  • Optional OpenAI API for complex reasoning

Security & Privacy

  • Your data stays on your network
  • JWT authentication with role-based access
  • Simple queries never leave your server
  • No training on your data by third parties
  • Soft-delete — no accidental data loss

Minimum Requirements

  • Ubuntu 22.04+ / any modern Linux
  • 16 GB RAM
  • NVIDIA GPU with 8+ GB VRAM
  • 50 GB storage

What's Included

  • Full web UI (React + TypeScript)
  • Backend API (FastAPI + PostgreSQL)
  • Local AI models via Ollama
  • CLI tool for power users
  • Admin dashboard (God Mode)
